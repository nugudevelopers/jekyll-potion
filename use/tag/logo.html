logo

logo 는 site 의 제목, logo 를 표현하는 tag 로서 layout 에서 사용됩니다.

물론 layout 파일에 직접 site 의 제목, logo 를 꾸밀 수 있지만, logo 를 사용할 경우 별도의 수고를 들이지 않아도 됩니다.

기본적인 선언은 다음과 같습니다.
