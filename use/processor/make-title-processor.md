# make_title_processor 

문서의 제목이 없을 경우 제목을 추출하고 문서의 제목, 설명을 HTML 에 포함시킵니다.

## Hook event

| owner   | event          | 설명                                                                                 |
|---------|----------------|------------------------------------------------------------------------------------|
| `:site` | `:post_read`   | 문서의 제목이 없을 경우 제목을 추출합니다. 정의된 제목이 없을 경우 문서에 가장 먼저 기술된 `#` ~ `###` 의 내용을 제목으로 간주합니다. |
| `:page` | `:post_render` | 추출한 제목, 설명을 HTML meta 태그로 추가합니다.                                                   |
